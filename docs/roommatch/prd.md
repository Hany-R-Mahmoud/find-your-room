# MVP PRD

## Product Thesis

RoomMatch helps expats and young professionals in Riyadh find a room and compatible household faster and more safely than fragmented listing channels.

## Primary Users

### Room seeker

- needs a safe, affordable move quickly
- wants clarity on total price, neighborhood, rules, and housemates
- values trust more than novelty

### Master tenant or small landlord

- needs qualified demand and fewer conflicts
- values verified applicants and clear expectations

## Core User Story

As a room seeker, I want to understand whether a room and a household actually fit my life before I commit to a move, so I can avoid scams, mismatch, and wasted time.

## Core Experience

1. User chooses `Find a room`, `Find a roommate`, or `Both`
2. User completes budget, move date, neighborhood, and lifestyle preferences
3. User sees a ranked room feed with fit explanations
4. User opens a room detail page with household and trust context
5. User requests a match or viewing
6. User tracks the path toward move-in through `My Stay`

## Success Metrics

- onboarding completion rate
- verification completion rate
- seekers receiving three or more relevant options
- request rate from listing detail
- mutual interest rate
- time to first qualified conversation
- room fill time
- fraud and dispute rate

## Product Principles

- trust before transaction
- explain the fit, do not hide it behind `AI`
- approximate location before exact address
- respectful and practical tone, never social-discovery tone
- bilingual ready from the start
